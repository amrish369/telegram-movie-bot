# 🎬 CineRadar AI Telegram Bot

A powerful Telegram Movie Bot with:

- ⚡ Instant streaming via file_id
- 🔍 Smart search (fuzzy matching)
- 🎬 OMDb API integration (posters + details)
- 📩 Movie request system
- 🔔 Auto notify users
- 🧠 Advanced filtering (language, quality, year)

## 🚀 Features

- Auto-delete movies after 2 min
- Channel join force system
- Admin dashboard (/stats, /broadcast)
- Auto GitHub sync
